# Code Symphony — Landing Page

Statischer OnePager für Code Symphony UG (haftungsbeschränkt).

## Lokal starten

Beliebigen HTTP-Server verwenden:

```bash
npx -y serve .
```

## Struktur

- `index.html` — OnePager
- `impressum.html` — Impressum (§ 5 DDG)
- `datenschutz.html` — Datenschutzerklärung (DSGVO)

## Rechtliches

- Impressum gemäß § 5 DDG
- Datenschutzerklärung gemäß DSGVO Art. 13/14
- Fonts lokal gehostet (kein Google CDN, vgl. LG München I, Az. 3 O 17493/20)
- Keine Cookies, kein Tracking
